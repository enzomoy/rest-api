Une API pour gérer de manière simple vos réservations !

## Installation

* npm
  ```sh
  npm install
  ```


* Création d'un fichier .env à la racine 
  ```sh
  DB_SERVER = 'localhost'
  DB_USER = 'user'
  DB_PASSWORD = 'mot-de-passe'
  DB_NAME = 'nom-db'
  JWT_SECRET = 'code-jwt'  
  ```
  
## Démarrage

  ```sh
  npm start
  ```
